To run in VS Code:
1. Install extension code runner
2. Go to Settings 
3. Type 'Code runner' into search bar
4. Enable 'Code Runner: Run in Terminal'

From the terminal:
1. compile code: gcc main.c map.c -o program
2. run code: ./program
